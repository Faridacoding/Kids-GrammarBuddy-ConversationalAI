# GrammarBuddy: Conversational Al Learning Agent

## The Problem:
Traditional grammar tools are dry, correction-heavy, and fail to keep children aged 7-10 engaged.

## The Product Vision:
A "flipped classroom" or Teacher/Student role-reversal interactive learning app where the child becomes the teacher, speaking to an AI student named Buddy who learns along with them in real time.

## Technical & Product Architecture:
* **Real-Time UX:** Built using React 19 and the Gemini Live API to achieve sub-second voice latency, preventing cognitive drop-off for young learners.  
* **Visual Engagement:** Orchestrated an image-generation pipeline to dynamically illustrate story scenes as the child narrates, turning verbal homework into a physical keepsake (illustrated book).  

## Product Edge Cases Solved
Handled voice-to-text filtering to handle stuttering and background noise common in child environments.
